# Exception Handling & Debugging Demo (Java + IntelliJ)

This project is designed for a live workshop teaching:
- Java exception handling (checked, unchecked, custom)
- Debugging with IntelliJ IDEA
- Backend-style examples: simulated DB operations, null checks, user operations

## How to Use
1. Clone repository
2. Open in IntelliJ IDEA
3. Run `App.java`
4. Follow live demo instructions (intentional bugs included)

## Key Learning Areas
- Handling SQLException (checked)
- Debugging a NullPointerException
- Creating custom exceptions
- Using breakpoints, stepping, watches, evaluate expression in IntelliJ
